#RNF01 - Todo e qualquer arquivo do projeto do software deve estar em UTF-8
#RNF02 - O dinheiro dos Financiadores deve ser devolvido em sua carteira caso o projeto atinja a data limite (sempre 6 meses após a criação do projeto) e não tenha atingido a meta
#RNF03 - A senha para criação de qualquer nova conta (Administrador, Funcionário e Financiador) deverá ter no mínimo 6 caracteres e no máximo 40
#RNF04 - Será feita verificação de formatação dos emails, sempre que forem requisitados pelo sistema, para assegurar a validez do mesmo
#RNF05 - Os campos de valores referentes a dinheiro no sistema, terão um intervalo de no mínimo 0,01 (0.01) e no máximo 1.000.000,00 (1000000000.00) 
#RNF06 - Os campos de valores referentes a dinheiro no sistema terão sempre duas casas decimais
#RNF07 - Nenhum campo de preenchimento de formulários no sistema poderá ser vazio (nulo)
#RNF08 - Os campos de preenchimento que requisitarem um número de CPF, serão sempre validados para que assegure-se um CPF existente
#RNF09 - O campo de preenchimento que requisitar "nome do projeto", na página de criação de um novo projeto, deverá ter no mímino 1 caractere e no máximo 100
#RNF10 - O campo de preenchimento Gêneros e Autor, na página de criação de um novo projeto, pode ter um ou mais nomes separados por vírgula
#RNF11 - O design do projeto deve ser feito utilizando o framework Bootstrap
#RNF12 - O design do projeto deve ser responsívo para atender todas as plataformas que rodem html/javascript
#RNF13 - O projeto deve ser feito utilizando a linguagem PHP para operações backend e criação das paginas front-end
#RNF14 - Qualquer usuário anônimo poderá se cadastrar como Financiador
#RNF15 - Somente administradores podem criar outra conta de Administrador
#RNF16 - Somente administradores podem criar uma conta de Funcionário
#RNF17 - Qualquer usuário cadastrado pode altearar seus dados pessoais no sistema
#RNF18 - Somente administradores podem criar projetos
#RNF19 - Somente Financiadores possuem carteira virtual e podem adicionar dinheiro à carteira
#RNF20 - Qualquer usuário pode listar todos os projetos
#RNF21 - Financiadores podem apoiar qualquer projeto com meta em aberto
#RNF22 - Administradores podem designar Funcionários a um projeto
#RNF23 - Todos podem fazer login
#RNF24 - Somente administradores podem editar projetos existentes
#RNF25 - Somente administradores podem excluir projetos existentes
#RNF26 - Qualquer usuário do sistema pode excluir sua própria conta
